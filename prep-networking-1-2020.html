Binary and IP Addresses

Last week you have already learned about IP addresses and how to connect your Raspberry Pi to the Internet. Now you will learn even more about IP addressing and the fundamental aspects that enable the Internet.

To know more about networking we will be using the book Computer Networking: A Top Down Approach, 7th edition. We will be using this book in the upcoming three weeks and in the course TTM4200 in the Spring semester, so please try to get a copy as soon as possible.

Reading the first chapter is recommended but not mandatory. Each week will focus on specific sections that need to be read and understood. Details about the required readings will be available every week, in bold.

Learning Goals

After this week you will be able to:

  • Understand and apply basic binary arithmetic
  • Understand IP addresses and their organisation into subnets
  • Create and configure Local Area Networks (LANs)
  • Use basic Linux commands for managing:
    • IP addresses (IPv4 and IPv6)
    • IP subnets and masks
