Passwords and Cryptography

This week, we will continue with passwords as topic and the newest trends in password creation and storage. Various techniques of password cracking will be introduced, as well as some of the password cracking countermeasures. We will also talk about common mistakes that some of us can fall into while storing their passcodes.

Learning Goals for this Week

By the end of this week, you will:

  • Learn the basics in password configurations, storage and cracking.
  • Get introduced to cryptography and to the main encryption algorithms.

And by completing the Lab, you will:

  • Learn how to crack a password
