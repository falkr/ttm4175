# cd


Moving into a directory:

```bash 
cd <path-to-directory>
```

Moving one directory up:

```bash
cd ..
```

Moving in the home directory:

```bash
cd ~
```

# curl

Download a file from the internet. 

```bash
curl https://www.ntnu.no/o/ntnu-theme/images/logo_ntnu.svg
```
