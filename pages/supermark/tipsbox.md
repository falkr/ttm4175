# Documentation

Paragraph style to show tips in a box.






# Example

* Leg vekt på å **forklare ting til hverandre**.
* Ikke bare "bli enig" om hvilken løsning som er best, men **forklar hvorfor**.
* Pass på at **alle deltar** og er med.
